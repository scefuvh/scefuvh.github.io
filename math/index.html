数列的极限

极限

定义:

定理

  • 有界性: $ 设{x_0}, \exists M\in R>0, 使\forall n \in N^+, |x_n| \leq M 则{x_n}为有界数列, M为一个界.$
  • 单调性: $ 设\lbrace x_0\rbrace, 对\forall n\in N^+, 有x_n \leq x_{n+1}(或x_n \geq x_{n+1}), 则称{x_n}单调增加(或单调减少). 如果是< 与 >则为严格单调增加(或严格单调减少). $

  • 有界性: 收敛数列有界(逆否命题: 无界数列发散)

  • 保序性: $设lim_{n \rightarrow \infty}{x_n} = a, lim_{n \rightarrow \infty}{y_n} = b, a > b, 则\exists N \in N^+, 当n > N时, 有x_n > y_n$
  • 保序性逆定理: $设lim_{n \rightarrow \infty}{x_n} = a, lim_{n \rightarrow \infty}{y_n} = b, 且\exists N \in N^+, 使x_n \geq y_n, 则a \geq b$

  • 单调数列必有极限.
  • 柯西准则: $数列{x_n}收敛的充要条件为对任意给定的a>0, \exists N \in N^+ 当n > N时, 对\forall p \in N^+, 存在|x^{n+p} - x^n| < a$
  • $\lim_{n \rightarrow \infty}\frac{a_n}{n} = \lim_{n\rightarrow \infty}\frac{a_n-a_{n-1} }{n-(n-1)}$

函数的极限

定义:类似数列, 不过”n > N”在$x_0$处的极限变为$|x - x_0| < N$
利用定义证明极限时, 若为$x \rightarrow x_0$的极限, 则需要利用放缩, 赋N为定值扽等方式将$|f(x) - A| < M$的式子凑成含有$|x - x_0|$结构的式子.

相关性质

  • 唯一性
  • 局部有界性
  • 局部保号性
  • 保序性逆定理: 若$\lim_{x \rightarrow x_0}f(x) = A, \lim_{x\rightarrow x_0}g(x) = B, f(x)\leq g(x)$则存在k > 0, 使得$x \in U^{\circ}(x_0, k)$时, 有 A $\leq$ B.
  • 保序性: 若$lim_{x \rightarrow x_0}f(x) = A, \lim_{x \rightarrow x_0}g(x) = B, A > B$, 则存在k > 0, 使得$x\in U^{\circ}(x_0, k)$时, f(x) > g(x)
  • 迫敛性: 如果$\exists a > 0$, 使得当$x \in U^{\circ}(x_0, a)$时, 有$f(x) \leq h(x) \leq g(x)$, 且$\lim_{x \rightarrow x_0}{f(x)} = \lim_{x \rightarrow x_0}{g(x)} = A$, 则$\lim_{x \rightarrow x_0}{h(x)} = A$

  • 柯西准则: f(x)极限存在的充要条件是$\forall a > 0, \exists b > 0, \forall x_1, x_2 \in U^{\circ}(x_0, a), |f(x_1) - f(x_2)| < b$

无穷大与无穷小

定义

注: 所有无穷大与无穷小都是相对某个变化而言的, 某些操作(如比较)必须发生在同一变化的无穷大(小)上.
无穷大: 极限为$\infty$的过程.
无穷小: 极限为0的过程.

运算性质

  • 有限个无穷小的和, 乘都是无穷小.
  • 无穷小与有界变量的乘积为无穷小.
  • 无穷小与极限不为0的变量的商是无穷小.
    证明:
    设$\lim_{x \rightarrow x_0}f(x) = 0, \lim_{x \rightarrow x_0}g(x) = b != 0$, b > 0, 根据局部有界性, 存在a > 0, 当x$\in U^{\circ}(x_0, a)时, 有g(x) > b/2 > 0$
    因此, 0 < 1/g(x) < 2/b, 则g(x)有界, 所以f(x)g(x)为一个有界量乘无穷小, 为0.

比较(以无穷小为例, 无穷大可以类比)

  1. 若$\lim{f(x)/g(x)} = 0$, 则称在这个过程中, f(x)是g(x)的高阶无穷小.
  2. 若$\lim{f(x) / g(x)} = \infty$, 则称在这个过程中, f(x)是g(x)的低阶无穷小.
  3. 若$\lim{f(x) / g(x)} = k, k为常数且\neq 0, 1$, 则称在这个过程中, f(x)是g)(x)的同阶无穷小.
  4. 若$\lim{f(x) / g^l(x)} = k, k \neq 0, l > 0$, 则称在这个过程中 f(x)是g)(x)的k阶无穷小.
  5. 若$\lim{f(x) / g(x)} = 1$, 则称在这个过程中, f(x)是g(x)的等价无穷小, 记作f(x) ~ g(x).
  6. 若$\lim{f(x) / g(x)}$不存在, 则f(x)与g(x)在这个过程中无法比较.
    求极限时, 可以对函数作同过程的等价无穷小替换, 但是注意替换后分子或分母不能变成0.

连续

定义: $lim_{x \rightarrow x_0}{f(x)} = f(x_0)$, 称f(x)在x = $x_0$这一点连续.
定义2: $lim_{\Delta x \rightarrow 0}{\Delta y} = 0$.

充要条件: 左极限等于右极限.
特殊情况—一致连续: $设f(x)在I上有定义, \forall a > 0, \exists b > 0, 对x_1, x_2 \in I, |x_1 - x_2| < a, 有|f(x_1) - f(x_2)| < b$, 则称f(x)在I上一致连续.

微分中值定理

定义

极大值: f(x)在U(l)有定义, 若存在x∈U(l), f(x) $\leq$ f(l), 则f(l)是最大值
极小值: f(x)在U(l)有定义, 若存在x∈U(l), f(x) $\geq$ f(l), 则f(l)是最小值
极值: 极大值和极小值.
极值点: 使f(x)为极值的x称为极值点.
驻点: 若$f^{‘}(x)=0$, 则x为驻点.

定理

费马引理
: 若f(x)在x=a处取极值且可导, 则$f^{‘}(a) = 0$

利用f(a) > f(x), 构造导数定义, 后讨论左导与右导正负性, 利用可导等价左导等于右导, 得导数为0.

罗尔定理
: 若f(x)在[a, b]连续, (a, b)可导f(a) = f(b)则存在h∈(a, b)令$f^{‘}(h)=0$


闭区间连续使用介值定理得出有max与min, 端点相等得到max > f(a) = f(b)和min < f(a) = f(b)必据其一, 则(a, b)存在一个极值, 由可导推出导数为0.

拉格朗日中值定理
: 若f(x)在[a, b]上连续, (a, b)可导, 则存在c∈(a, b)使$f^{‘}(c) = “f(b) - f(a)” / “b - a”$

利用罗尔定理, 希望端点相等, 故考虑转动坐标系, 则得出g(x) = f(x) - “f(b) - f(a)” / “b - a” * (x - a), 然后利用罗尔定理证明

柯西中值定理
: 若f(x), g(x)满足[a, b]连续, (a, b)可导, g^{‘}(x) != 0, 则存在l∈(a, b) 使得f^{‘}(x) / g^{‘}(x) = “f(b) - f(a)” / “g(b) - g(a)”

将拉格朗日证明中的x, b, a换成g(x), g(b), g(a)

泰勒中值定理
: 若f(x)在x-=$x_0$处n+1阶可导, 则有$f(x) = f(x_0) + f^{(1)}(x_0)(x - x_0) + \frac{f^{(2)}(x_0)(x - x_0)^2}{2!} + … + \frac{f^{(n)}(x_0)(x - x_0)^n}{n!} + \frac{f^{(n+1)}(a)(x - x_0)^{n+1} }{(n+1)!}$

泰勒公式

函数是无穷维的向量
线性组合: 将一个一般的向量表现为一组简单的向量的求和.

两种级数
傅立叶级数: $f(x) = c \times 1 + a_1sinx + a_2sin(2x)+……+a_nsin(nx) + b_1cos(x) + b_2cos(2x)+……+b_ncos(nx)$
幂级数: $f(x) = a_0 \times 1 + a_1(x - x_0) + a_2(x - x_0)^2 + …… + a_n(x - x_0)^n + ……$

解决三个问题, 得出泰勒公式

  1. 表示成幂级数的条件是什么?
    f(x)有$x - x_0$处可导, $f(x) = f(x_0) + A(x - x_0) + o[x - x_0]$

  2. 确定$a_0, a_1, a_2, …….$
    取$x = x_0$, 则$a_0 = f(x_0)$
    由幂级数的公式, 发现如果求一阶导, 则$a_0会消失, a_2, a_3, ……还会剩下(x - x_0)的结构使得取值后被消掉, 只剩下a_1$
    则$a_1 = f^{‘}(x_0)$
    由此得到$a_n = f^{(n)}(x_0) / n! $

  3. “+…”如何表示?
    皮亚诺公式 若f(x)在U($x_0$)中n阶可导, 则在此邻域内有f(x) = $f(x_0) + f’(x_0)(x - x_0) + “f’’(x_0)(x - x_0)” / 2! + … + f^{(x)(x_0)(x - x_0) / n!} + o[(x - x_0)^n] $
    证明:
    构造R(x) = f(x) - $(f(x_0) + f^{(1)}(x_0)(x - x_0) + “f^{(2)}(x_0)(x - x_0)” / 2! + … + f^{(x)}(x_0)(x - x_0) / n! + o[(x - x_0)^n] )$
    则只要证明$ lim_{x \rightarrow x_0}{R(x_0) / (x - x_0)^n} = 0$
    易知R(x)及n阶导数都为0
    则对极限用n-1次洛必达法则, 得原式 = $lim_{x \rightarrow x_0}{R^{(n-1)}(x) / “n!(x - x_0)”}$
    注意: 因为R(x)的n阶可导只能得出n-1阶连续, 则再用洛必达法则无法求出极限.
    此时应用柯西中值定理: $lim_{x \rightarrow x_0}{R^{(n-1)}(x) / “n!(x - x_0)”}$ = $”lim_{x \rightarrow x_0}{R^{(n-1)}(x) - R^{(n - 1)}(x_0)”/ “n!(x - x_0) - 0”}$ = $R^{(n)}(x_0) / n!$ = 0

相关定理

平面

平面法向量为$\vec{s}(x_0, y_0, z_0)$
设点A($x_1$, $y_1$, $z_1$), B($x_2$, $y_2$, $z_2$), C($x_3$, $y_3$, $z_3$)在平面上.

平面的点法式方程: $x_0(x - x_1) + y_0(y - y_1) + z_0(z - z_1) = 0$

平面的一般方程: $x_0x + y_0y + z_0z + D = 0$
当D = 0时, 平面过原点.
当$x_0$ = 0时, 平面与x轴平行(D $\ne$ 0)或通过x轴(D = 0), y, z以此类推.

平面的三点式方程:
$\left|
\begin{matrix}
x - x_1 & y - y_1 & z - z_1 \ x_2 - x_1 & y_2 - y_1 & z_2 - z_1 \ x_3 - x_1 & y_3 - y_1 & z_3 - z_1
\end{matrix}
\right|$ = 0

平面束方程: 设直线l为
$
\begin{cases}
A_1x + B_1y + C_1z + D = 0 \ A_2x + B_2y + C_2z + D = 0
\end{cases}
$
则所有过l的平面为$k(A_1x + B_1y + C_1z + D) + n(A_2x + B_2y + C_2z + D) = 0$, k, n 为任意常数.
如果两个面平行, 则该方程为与他们平行的所有平面.

平面F(x, y, z) = 0在M($x_0, y_0, z_0$)切平面的法向量和法线的方向向量为($F_x(x_0, y_0, z_0), F_y(x_0, y_0, z_0), F_z(x_0, y_0, z_0)$)

直线

设点M($x_0$, $y_0$, $z_0$), N($x_1$, $y_1$, $z_1$)在直线l上.
直线的方向向量为(m, n, p)

点向式方程为: $\frac{x-x_0}{m} = \frac{y-y_0}{n} = \frac{z-z_0}{p}$. 若m = 0, 则方程为x = $x_0$; $\frac{y-y_0}{n} = \frac{z-z_0}{p}$

参数方程: x = $x_0$ + mt; y = $y_0$ + nt；z = $z_0$ + pt

两点式方程: $\frac{x - x_1}{x_0 - x_1} = \frac{y - y_1}{y_0 - y_1} = \frac{z - z_1}{z_0 - z_1}$

由于直线可看作两个平面的交线, 故直线方程可写成两个平面方程的联立, 直线的方向向量两个平面的法向量的叉乘.

直线的切线与法平面: 直线的参数方程在p($x_0, y_0, z_0$)的x, y, z的导数就是p点切线的方向向量和法平面的法向量. 做题时, 可以令x = t求y(x)和z(x)的偏导.

定义与性质

设数轴上的点集E和任意一点P.

内点
: $\exists$a>0, U(P, a)$\subset$E, 则P是E的内点.

外点
: $\exists$a > 0, U(P, a)$\bigcap$E = $\emptyset$, 则P是E的外点.

边界点
: $\forall$a > 0, U(P, a)$\bigcap$E $\neq\emptyset 且两集合没有包含关系$, 则P是E的边界点.

聚点
: $\forall$a > 0, $U^{0}(P, a)\bigcap E\neq\emptyset$, 则P是E的聚点.

孤立点
: $\exists$a > 0, U(P, a)$\bigcap$E = {P}, 则P是E的孤立点.

开集
: 所包含的每个点都是内点的点集.

连通
: 若E中任意两点都可以用属于E的若干条直线段连接, 则E连通.

开区域
: 连通的开集.

闭区域
: 开区域及边界.

有界集
: $\exists$k > 0, E$\subseteq$U(O, k), O是原点, 则E为有界集.

无界集
: 不存在有界集中的k的点集.

多元函数的极限
: 设f(x, y)的定义域为D, P是其聚点, 若$\forall$ a > 0, $\exists$b, 使得0 $<$ |$PP_0$| = $\sqrt{(x - x_0)^2 + (y - y_0)^2}$ $<$ b的一切点都有|f(x, y) - A| $<$ D成立, 则A是f(x, y)当x$\rightarrow x_0$, y$\rightarrow y_0$时的极限. P的趋进方式应该是任意的.

累次极限

连续
: 设f(P)的定义点集为D, 若$P_0$是D的聚点且$\lim_{P\rightarrow P_0}f(P) = f(P_0)$

多元函数拥有介值定理, 一致连续定理, 运算复合连续等一元函数拥有的所有定理, 只是将区间换成区域

思路: 因为I直接积分不好积分, 可以将这个一元定积分用平方形式换成二元然后换元.
$I^2 = \int_{-\infty}^{\infty}e^{-x^2}dx\int_{-\infty}^{\infty}e^{-y^2}dy = \int_{-\infty}^{\infty}e^{-x^2 - y^2}dxdy$
令x = rcosa, y = rsina.
原式 = $\int\int_De^{-r^2}rdrda = \int_{0}^{2\pi}da\int_{0}^{\infty}e^{-r^2}rdr = \pi$
$I = \sqrt{\pi}$

极坐标系中的$rdrd\theta =$直角坐标系下的dxdy
同理, 球坐标中的$r^2sin\varphi drd\theta d\varphi =$ 直角坐标系下的dxdydz

曲线积分

边界曲线的正向即围成的区域在左手时的方向.

路径无关
: 如果区域内任意两条共起止点的光滑有向曲线存在曲线积分相等或任意闭合权限积分为0, 则称那个积分与路径无关.

定义
: f(x, y)在曲线上有定义, 将曲线等分成n段, 每段取一个点(a, b)计算f(a, b), 如果f(a, b)乘上曲线长度的累加和

存在条件
: 当f(x ,y)在光滑曲线L上连续时$\int_Lf(x, y)ds$存在

计算定理:

第一类
: 若f(x, y)在L上有定义连续, 设L = $\begin{cases}
x = g(t) \ y = h(t)
\end{cases} t\in[a, b]$
$则\int_Lf(x, y)ds = \int_a^bf[g(t), h(t)]\sqrt{g^{2}(t) + h^{2}(t)}dt$对三元函数类似, 记得上限始终大于下限.(若某个变量为一个定值, 他的导数视为1, 若没有, 导数视为0)

第二类
: 本质上就是对向量积的积分转换成对坐标积分的和, 和第一类一样有积分曲线是参数方程的计算法, 不过上线可以小于下限.

设开区域G是个单联通域, f(x, y), g(x, y)在G内有一阶连续偏导数, 则

  1. $\int_Lfdx + gdy$在G内路径无关
  2. $\exists h(x, y)\in G, dh = fdx + gdy$
  3. $\forall (x, y)\in G, \frac{\vartheta f}{\vartheta y} = \frac{\vartheta g}{\vartheta x}$

格林公式
: 设闭区域D由分段光滑曲线L围成, f(x, y)和g(x, y)在D上有一阶连续偏导数, 则
$\iint_D(\frac{\vartheta f}{\vartheta x} - \frac{\vartheta g}{\vartheta y})dxdy = \oint_L(gdx + fdy)$

曲面积分

曲面投影问题: 在有向曲面a上取一小块$\Delta S$, 其在xoy面上的投影($\Delta S_{xy}$)当cosd > 0时为正, $<$ 0时为负, 等于0时为0. d为a的法向量与z正半轴夹角.

定义: 光滑曲面a, f(x, y, z)在a上有界. 把a分成n块, 取点, 乘上小块面积, 作和. 若n趋向无穷时极限存在, 则称其为f在a上对面积的曲面积分.

存在条件: 被积函数在光滑曲面上连续.

计算公式:

第一型
: 若曲面a = $\begin{cases} x = g(t) \ y = h(t) \ z = u(t) \end{cases}, \iint_af(x, y, z)dS = \int_k^m f(g(t), h(t), u(t))\sqrt{g^{‘2}(t) + h^{‘2}(t) + u^{‘2}(t)}dt$. 如果变成二重积分, 积分面积就是曲面对两个自由变量的代表的平面的投影, 根号里面就是被表示的那个变量作函数对三个变量导数的平方和(注意根号部分不能恒为0)

第二型
: 本质上就是函数对有向积分曲面的某一个面投影积分, 对哪个双变量面投影, 就用那两个变量做积分变量, 其余同第一型, 注意正负号.

注意, 曲面积分不遵守对称计算规律

两型转换公式
: $\iint_a Pdydz + Qdzdx + rdxdy = \iint_a(Pcosb + Qcosc + Rcosd)dS$, 其中P, Q, R分别为x, y, z坐标的坐标函数, b, c, d分别为积分曲面对x, y, z轴的方向角, S为积分曲面的面积.

曲面投影到积分曲面上应该对每个点做的垂线只有一个交点, 如果不满足需要分别求

$dxdy = \frac{dydz}{-z_x} = \frac{dxdz}{-z_y}$

高斯公式
: 设空间闭区域a由分片光滑的外测闭曲面b围成, f(x, y, z), g(x, y, z), h(x, y, z) 在a上有一阶连续偏导数, 则$\iiint_a(\frac{\vartheta f}{\vartheta x} + \frac{\vartheta g}{\vartheta y} + \frac{\vartheta h}{\vartheta z})dV = \oiint fdydz + gdzdx + hdxdy$

级数

若$\sum_{i=1}^{\infty}a_i$收敛, 则$\sum_{i=k+1}^\infty a_i$收敛.

级数加括号后收敛性不变. 但是去掉括号则不一定.

级数收敛$\rightarrow \lim_{n\rightarrow\infty}a_n = 0$

柯西收敛准则
: $\sum_{i=1}^\infty a_i$收敛的充要条件是$\forall k > 0, \exists N, 当n > N时, \forall p\in N, |\sum_{i=n+1}^{n+p}a_i| < k$

比较判别法
: 设$\sum_{n=1}^\infty a_i$和$\sum_{n=1}^\infty b_i$都是正常级数, $\exists N\in N, k\forall n > N ka_n > b_n$, 则若$\sum_{n=1}^\infty a_i$收敛, $\sum_{n=1}^\infty b_i$收敛.

极限判别法
: 设$\sum_{n=1}^\infty a_i$和$\sum_{n=1}^\infty b_i$都是正项级数, 若$\lim_{n\rightarrow\infty}\frac{a_n}{b_n} = l$, 则:

  1. 0 $<$ l $< +\infty$, 两级数敛散性相同.
  2. l = 0, b收敛a收敛.
  3. l = $+\infty$, b发散a发散.

比式判别法/D’Alembert判别法
: $\sum_{n=1}^\infty a_n$是正项级数, 若$\lim_{n\rightarrow\infty}\frac{u_{n+1} }{u_n} = k$, 可充分不必要地得出k $<$ 1收敛, > 1发散, = 1无法使用.

根式判别法/柯西判别法
: $\sum_{n=1}^\infty a_n$是正项级数, 若$\lim_{n\rightarrow\infty}\sqrt[n]{a_n} = p$, 则 $<$ 1收敛, $>$ 1发散, =1无法使用.

莱布尼茨定理
: 如果交错级数满足$a_n\geq a_{n+1}\land lim_{n\rightarrow\infty}a_n = 0$则级数收敛, 和$\leq a_1$, 其余项|$r_n$|$\leq a_{n+1}$

若$\sum_{n=1}^\infty|a_n|$收敛, 则$\sum_{n=1}^\infty a_n$收敛

绝对收敛
: $\sum_{n=1}^\infty|a_n|$收敛

条件收敛
: $\sum_{n=1}^\infty|a_n|$发散, $\sum_{n=1}^\infty a_n$收敛, 则称其为条件收敛

函数项级数

收敛点
: 如果对于$x_0\in I$, 数项级数$\sum_{n=1}^\infty f_n(x_0)$收敛, 则$x_0$为此级数的收敛点

发散点
: 如果对于$x_0\in I$, 数项级数$\sum_{n=1}^\infty f_n(x_0)$发散, 则$x_0$为此级数的发散点

收敛域
: 收敛点组成的域.

发散域
: 发散点组成的域.

和函数
: 收敛域上, 函数项级数的和

幂级数
: $\sum_{n=0}^\infty a_n(x - x_0)^n$的级数称为幂级数.

Abel定理
: 如果幂级数在$x=x_0$处收敛, 则他在满足不等式x$<|x_0|$的一切x处绝对收敛, 若发散, 则它在$x>|x_0|$处都发散

收敛半径
: 幂级数中恰好处于收敛与发散域界上的正数R

若幂级数所有系数!=0, 设$\lim_{n\rightarrow\infty}\frac{a_{n+1} }{a_n} = p$或$lim_{n\rightarrow\infty}\sqrt[n]{a_n} = p$, 则Rp = 1

幂级数和函数在收敛区间内连续, 可积, 无限次可导, 且极限, 积分与求导运算符可以与求和号交换.

部分和函数的结果

幂级数展开

幂函数展开与泰勒函数展开的不同:

  1. 泰勒函数展开是某点上的, 而幂函数展开是整个函数上的.
  2. 泰勒函数展开要求函数在那个点无穷次可导, 幂函数不要求.
  3. 泰勒函数展开只能保证在展开点等于函数本身, 幂函数一定等于函数本身.
  4. 泰勒函数展开有公式可循, 幂函数只能依靠已知展开推导.

函数的幂函数展开就是函数的泰勒函数.

泰勒级数

f(x)的展开式的$a_n = \frac{f(x_0)}{n!}$

f(x)在$x_0$处的泰勒级数, 在U($x_0$)收敛于f(x)$\i$在U($x_0$)内$\lim_{n\rightarrow\infty}R_n(x) = \frac{f^{(n+1)(k)} }{(n+1)!}(x-x_0)^{n+1} = 0$(泰勒公式在$x_0$处的拉格朗日余项)

$\exists M > 0, \forall x\in U(x_0), f^{(n)}\leq M$, 则f(x)在U($x_0$)内可展开成点$x_0$的泰勒级数.

三角级数

简谐振动函数
: $f(t) = A_0 + \sum_{n=1}^\infty A_nsin(nwt + \varphi_n)$

三角级数
: 令$\frac{a_0}{2} = A_0, a_n = A_nsin\varphi_n, b_n = A_ncos\varphi_n, wt = x. \frac{a_0}{2} + \sum_{n=1}^\infty(a_ncosnx + b_nsinnx)$

三角函数系
: 1, cosx, sinx, cos2x, sin2x, $\cdots$ cosnx, sinnx, $\cdots$

三角函数系里任意了两个不同函数成绩的积分在[$-\pi$, $\pi$]上积分为0, 即三角函数系具有正交性.

若周期为2k,

f(x)展开的三角级数为$f(x) = \frac{a_0}{2} + \sum_{n=1}^\infty(a_ncos(\frac{\pi}{l}nx) + b_nsin(\frac{\pi}{l}nx))$

欲求$a_n$, 将两边乘上cosnx后同时积分, 得到$a_n = \frac{\int_{-l}^l f(x)cos(\frac{\pi}{l}nx)dx}{l}$

欲求$b_n$, 将两边乘上sinnx后同时积分, 得到$b_n = \frac{\int_{-l}^l f(x)sin(\frac{\pi}{l}nx)dx}{l}$

当k = $\pi$时,

$f(x) = \frac{a_0}{2} + \sum_{n=1}^\infty(a_ncosnx + b_nsinnx)$

欲求$a_0$, 将两边同时积分, 得到$a_0 = \frac{\int_{-\pi}^\pi f(x)dx}{\pi}$

欲求$a_n$, 将两边乘上cosnx后同时积分, 得到$a_n = \frac{\int_{-\pi}^\pi f(x)cos(nx)dx}{\pi}$

欲求$b_n$, 将两边乘上sinnx后同时积分, 得到$b_n = \frac{\int_{-\pi}^\pi f(x)sin(nx)dx}{\pi}$

奇函数的三角级数一定只含有sin项.
偶函数的三角级数一定只含有cos项.

Dirichlet 收敛定理

设f(x)是以2$\pi$为周期的周期函数或只在[$-\pi, \pi$]定义的函数(可以通过沿拓变成周期函数), 如果它满足:

  • 在一个周期内没有或有限个可去或跳跃间断点.
  • 在一个周期内没有或有限个极值点.

则f(x)的的三角级数收敛, 且:

  • $\forall x_0是间断点$, 级数收敛于$\frac{\lim_{x\rightarrow x_0^+}f(x)+\lim_{x\rightarrow x_0^-}f(x)}{2}$.
  • $\forall x_0是连续点$, 级数收敛于f($x_0$).

微分方程

微分方程中任意常数个数与阶数相同.

一阶线性齐次方程

形式: $\frac{dy}{dx} + P(x)y = 0$
通解: $y = Ce^{-\int P(x)dx}$

可化为齐次的方程

对$\frac{dy}{dx} + P(x)y = \frac{ax + by + c}{a_1x + b_1y + c_1}$, 可令X = x + k, y = y + h, 使得ah + bk + c = 0, $a_1h + b_1k + c_1 = 0$. 则可化成齐次方程求解.
如果h, k无解, 则a, b成比例, 可提出因子换元.

一阶线性非齐次方程解法

形式: $\frac{dy}{dx} + P(x)y = Q(x)$
通解: $y = (\int Q(x)e^{\int P(x)dx}dx + C)e^{-\int P(x)dx}$

Bernoulli方程

形式: $\frac{dy}{dx} + P(x)y = Q(x)y^n, n\neq 1$
通解: $y^{1-n} = (\int (1-n)Q(x)e^{\int (1-n)P(x)dx}dx + C)e^{-\int(1-n)P(x)dx}$

全微分方程

形式: $0 = P(x, y)dx + Q(x, y)dy$
通解: $u(x, y) = C$

注意, 全微分方程必须曲线积分与路径无关

积分因子法

形式: 设u(x, y)$\neq$0连续可微, 若u(x, y)P(x, y)dx + u(x, y)Q(x, y)dy = 0是全微分方程, 则u(x, y)是方程的积分因子

常见的常微分表达式

  1. $xdx + ydy = d\frac{x^2 + y^2}{2}$
  2. $\frac{xdy - ydx}{x^2} = d\frac{y}{x}$
  3. $\frac{xdy - ydx}{x^2 + y^2} = darctan\frac{y}{x}$
  4. $\frac{xdy + ydx}{xy} = dlnxy$
  5. $\frac{xdx + ydy}{x^2 + y^2} = d\frac{ln(x^2+y^2)}{2}$
  6. $\frac{xdy - ydx}{x^2 - y^2} = d\frac{1}{2}ln\frac{x+y}{x-y}$
