Electornmagnetism

静电场

相关定义

静电场
: 由静止的电荷产生的电场.
电偶极子
: 相隔一定距离的等量异号点电荷, 当它们的距离l比从它们到所讨论的场点的距离小得多时, 这个电荷系统称为电偶极子.
面在点乘叉乘运算中的方向由其的法向量代表.
闭合曲面的法向量方向由内向外.
电势梯度(U)
: 任意点P的各个方向中电势升高最快的方向和那个变化率.

相关公式

真空库伦定律: $F_{21} = \frac{kq_1q_2}{r^2_{21} }\hat{r_{21} }$, 其中$k = \frac{1}{4
\pi\varepsilon_0}$, k = $9 \times 10^9 N\cdot m^2/C^2, \varepsilon_0 = 8.85\times 10^{-12}C^2/N\cdot m^2$
电偶极距(p): p = ql. q为电偶极子中的一个的带电量.
电偶极子较远处的电场: $E = \frac{-p}{4\pi\varepsilon_0}r^3$, r为电偶极子中心到那个点的距离.
无限长的带电直线周围的电场: E = $\frac{\lambda}{2\pi\varepsilon_0x}, \lambda为线电荷密度$.
电通量($\phi$): $d\phi = E\cdot dS$, 电通量同时等于S内应画的电场线条数.
电势差(U): $U_{P_0P} = \int^{(P_0)}_{(P)}{\hat{ E}\cdot d\hat{l} } = W_{P_0P}/q$, l为$P_0$到P的矢量
电荷连续体的电势: $U = \int{\frac{dq}{4\pi\varepsilon_0r} }$, 其中r为考虑电势的点到被看成的点电荷的距离.
静电平衡高斯定律: 设E为静电平衡的物体表面 紧邻处的电场强度, $\theta$为附近的面电荷密度, $\theta = E\varepsilon_0$
电容(U): U = dq/dV.
电容串联时, 总电容1/U = $\sum_{i=1}{\frac{1}{U_i} }$
电容并联时, 总电容C = $\sum_{i=1}{C_i}$

相关定律

  • 高斯定律: 真空静电场中, 通过任意封闭曲面的电通量等于该封闭面所包围的电荷的电量的代数和/$\varepsilon_0$.
  • 平面定律: 在无限大的均匀带电平面附近, 电场匀强, 等于电荷密度/2$\varepsilon_0$.
  • 保守性: 静电场是保守场.
  • 静电平衡曲面定律: 孤立的静电平衡的导体的面电荷密度与曲率正相关.
  • 串联电容中$C_t = \sum{C_i}$, 串联电容中$\frac{1}{C_t} = \sum{frac{1}{C} }$

电流

相关定义

稳恒电流
: 导体内各处J不变的电流.
稳恒电场
: 不随时间改变的电场.

相关公式

电流强度(I): $I = \frac{\Delta q}{\Delta t}$
电流密度(J): $dI = J\times dS$, 方向为电流方向, S为通过的横截面积.
电源电动势($U$): $U = \frac{W_非}{q} = \int^+_-{E_kdl}, E_k$为外力带来的外电场, 若外力遍布回路,需做环路积分.
非静电性场强($E_k$): $E_k = \frac{\hat{F} }{q}$
电子载流子公式: J = nqv, v是通电时的漂移速度, q是单个载流子的电荷量.
霍尔电势差: U = $\frac{BI}{d}R_H, R_H$是霍尔系数, I是导体中的电流, d为导体有电势差面的宽度.
电阻(R): $dR = \frac{l}{dS\theta}$ 其中$\theta$是材料电导率.
微分欧姆定律: $J = (E+E_k)\theta$
电导率温变公式: $\frac{1}{\theta} = \frac{1+kt}{\theta_0}$ k为一个固定常数.
回路电流公式: $\sum\pm U$
稳恒电路欧姆定律: $\sum{\pm U} + \sum{\pm I_iR_i} = 0$, 其中与基准电路同向取-, 反向取+.

相关定律

  • 稳恒电流产生稳恒电场.
  • 流过封闭曲面的稳恒电流为0.
    • 流出节点的电流等于流入节点的电流.
  • 稳恒电场中沿闭合回路一周的电视差为0.

磁场

相关定义

真空磁导率($u_0$): $u_0 = 4\pi\times 10^{-7}N/A^2$
感应电场: 有旋(电场线类似涟漪)无源场, 电场线闭合, 电通量为0, 环路有电流.

相关公式

磁力: $F = qv\times B$
安培力: $dF = Idl\times B$
洛仑兹力: $F = qv\times B$
磁通量($\Phi$): $\Phi = \oint_S{B\cdot ds} = \iint_S{\hat{B}\cdot d\hat{S} }$, 磁通量可以直接相加.
带电粒子在磁场中的运动螺距(k): $k = Tv_h, v_h$是平行于磁感线的速度大小.
批萨定律: $dB = \frac{u_0}{4\pi} - \frac{Idl\times\hat{r} }{r^2}, $
无限长电导线对距离r处的点产生的磁场: $B = \frac{u_0I}{2r\pi}$
无限长通电螺线管的内部轴线上的磁场强度: $B = u_0nI$, n是匝数
通电螺线管端口中心的磁场强度: $B = \frac{u_0nI}{2}$
电磁感应: dU = $=\frac{d\Phi}{dt} = \hat{v}\times\hat{B}\cdot l$
电磁感应电荷: $q_i = -\frac{\Phi_2 - \Phi_1}{R}$
电磁感应等效非静电性场强($E_k$): $U_{动} = \oint{\hat{E_k}\cdot d\hat{l} }$
动生电动势($U_{ab}$): $U_{ab} = \int^b_a{\hat{v}\times\hat{B}\cdot d\hat{l} }$, 意义为一定时间内扫过面积的磁通量.
感生电动势($U_{感}$): $U_{感} = \frac{-d\Phi}{dt} = \oint_L{\hat{E}_i\cdot d\hat{l} } = -\iint_S{\frac{\partial\hat{B} } }{\partial t}\cdot d\hat{S}$, 因此感应电场与磁场增加方向呈左手螺旋关系(电场是四指).

闭合电路在匀强磁场中

磁距($P_m$): $P_m = SI\hat{n}, \hat{n}$是闭合电路的法线.
通电线圈在磁场中受到的力矩: $M = P_m\times B$
圆形通电线圈在轴线上的磁场: $B = \frac{u_0p_m}{2\pi(R^2 + x^2)^{\frac{3}{2} } }$, x为轴线上点到圆心的距离.

相关定律

磁场连续定理: 任意磁场中封闭曲面的磁通量为0.
动感定律: 动生电动势与感生电动势可以代数相加.
