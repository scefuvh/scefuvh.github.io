数列的极限

极限

定义:
几何解释:

定理

  • 有界性: $ 设{x_0}, \exists M\in R>0, 使\forall n \in N^+, |x_n| \leq M 则{x_n}为有界数列, M为一个界.$
  • 单调性: $ 设\lbrace x_0\rbrace, 对\forall n\in N^+, 有x_n \leq x_{n+1}(或x_n \geq x_{n+1}), 则称{x_n}单调增加(或单调减少). 如果是< 与 >则为严格单调增加(或严格单调减少). $
  • 唯一性: 若${x_n}$收敛, 则极限唯一.
  • 有界性: 收敛数列有界(逆否命题: 无界数列发散)
  • 保号性: $设lim_{n \rightarrow \infty}a_n = a, 则\exists N \in N^+, 当n > N.$
  • 保序性: $设lim_{n \rightarrow \infty}{x_n} = a, lim_{n \rightarrow \infty}{y_n} = b, a > b, 则\exists N \in N^+, 当n > N时, 有x_n > y_n$
  • 保序性逆定理: $设lim_{n \rightarrow \infty}{x_n} = a, lim_{n \rightarrow \infty}{y_n} = b, 且\exists N \in N^+, 使x_n \geq y_n, 则a \geq b$
  • 四则运算: 同数的四则运算.
  • 单调数列必有极限.
  • 柯西准则: $数列{x_n}收敛的充要条件为对任意给定的a>0, \exists N \in N^+ 当n > N时, 对\forall p \in N^+, 存在|x^{n+p} - x^n| < a$
  • $\lim_{n \rightarrow \infty}\frac{a_n}{n} = \lim_{n\rightarrow \infty}\frac{a_n-a_{n-1} }{n-(n-1)}$

函数的极限

定义:类似数列, 不过”n > N”在$x_0$处的极限变为$|x - x_0| < N$
利用定义证明极限时, 若为$x \rightarrow x_0$的极限, 则需要利用放缩, 赋N为定值扽等方式将$|f(x) - A| < M$的式子凑成含有$|x - x_0|$结构的式子.

相关性质

  • 唯一性
  • 局部有界性
  • 局部保号性
  • 保序性逆定理: 若$\lim_{x \rightarrow x_0}f(x) = A, \lim_{x\rightarrow x_0}g(x) = B, f(x)\leq g(x)$则存在k > 0, 使得$x \in U^{\circ}(x_0, k)$时, 有 A $\leq$ B.
  • 保序性: 若$lim_{x \rightarrow x_0}f(x) = A, \lim_{x \rightarrow x_0}g(x) = B, A > B$, 则存在k > 0, 使得$x\in U^{\circ}(x_0, k)$时, f(x) > g(x)
  • 迫敛性: 如果$\exists a > 0$, 使得当$x \in U^{\circ}(x_0, a)$时, 有$f(x) \leq h(x) \leq g(x)$, 且$\lim_{x \rightarrow x_0}{f(x)} = \lim_{x \rightarrow x_0}{g(x)} = A$, 则$\lim_{x \rightarrow x_0}{h(x)} = A$
  • 四则运算: 与数的四则运算相同.
  • 柯西准则: f(x)极限存在的充要条件是$\forall a > 0, \exists b > 0, \forall x_1, x_2 \in U^{\circ}(x_0, a), |f(x_1) - f(x_2)| < b$

无穷大与无穷小

定义

注: 所有无穷大与无穷小都是相对某个变化而言的, 某些操作(如比较)必须发生在同一变化的无穷大(小)上.
无穷大: 极限为$\infty$的过程.
无穷小: 极限为0的过程.

运算性质

  • 有限个无穷小的和, 乘都是无穷小.
  • 无穷小与有界变量的乘积为无穷小.
  • 无穷小与极限不为0的变量的商是无穷小.
    证明:
    设$\lim_{x \rightarrow x_0}f(x) = 0, \lim_{x \rightarrow x_0}g(x) = b != 0$, b > 0, 根据局部有界性, 存在a > 0, 当x$\in U^{\circ}(x_0, a)时, 有g(x) > b/2 > 0$
    因此, 0 < 1/g(x) < 2/b, 则g(x)有界, 所以f(x)g(x)为一个有界量乘无穷小, 为0.

比较(以无穷小为例, 无穷大可以类比)

  1. 若$\lim{f(x)/g(x)} = 0$, 则称在这个过程中, f(x)是g(x)的高阶无穷小.
  2. 若$\lim{f(x) / g(x)} = \infty$, 则称在这个过程中, f(x)是g(x)的低阶无穷小.
  3. 若$\lim{f(x) / g(x)} = k, k为常数且\neq 0, 1$, 则称在这个过程中, f(x)是g)(x)的同阶无穷小.
  4. 若$\lim{f(x) / g^l(x)} = k, k \neq 0, l > 0$, 则称在这个过程中 f(x)是g)(x)的k阶无穷小.
  5. 若$\lim{f(x) / g(x)} = 1$, 则称在这个过程中, f(x)是g(x)的等价无穷小, 记作f(x) ~ g(x).
  6. 若$\lim{f(x) / g(x)}$不存在, 则f(x)与g(x)在这个过程中无法比较.
    求极限时, 可以对函数作同过程的等价无穷小替换, 但是注意替换后分子或分母不能变成0.

连续

定义: $lim_{x \rightarrow x_0}{f(x)} = f(x_0)$, 称f(x)在x = $x_0$这一点连续.
定义2: $lim_{\Delta x \rightarrow 0}{\Delta y} = 0$.
本质: f可与$\lim$互换.
充要条件: 左极限等于右极限.
特殊情况—一致连续: $设f(x)在I上有定义, \forall a > 0, \exists b > 0, 对x_1, x_2 \in I, |x_1 - x_2| < a, 有|f(x_1) - f(x_2)| < b$, 则称f(x)在I上一致连续.

间断点

$\lim_{x \rightarrow x_0^+}f(x)$和$\lim_{x \rightarrow x_0^-}f(x)$都存在, 为第一类间断点.
若$\lim_{x \rightarrow x_0^+}f(x) = \lim_{x \rightarrow x_0^-}f(x)$$\neq f(x_0)$, 为可去间断点.
若$\lim_{x \rightarrow x_0^+}f(x) \neq \lim_{x \rightarrow x_0^-}f(x)$, 为跳跃间断点.
$\lim_{x \rightarrow x_0^+}f(x)$和$\lim_{x \rightarrow x_0^-}f(x)$不都存在, 为第二类间断点.
若其中一个极限$\rightarrow \infty$, 为无穷间断点.
若类似$sin(1/x)$这样不停震荡的, 为震荡间断点.

初等函数都是连续函数.
连续函数的初等变换结果仍是初等函数.

介值定理: 若f(x)在[a, b]连续, $f_{max}(x) = M$, $f_{min}(x) = m$, $\forall c \in [m, M], \exists a, f(a) = c$.

导数

定义

导数: 若f(x)在u(l, a)内有意义且极限lim($\Delta$x->0, $\Delta$y/$\Delta$x) = lim($\Delta$x->0, “f(l + $\Delta$x) - f(l)” / $\Delta$x) exists.
左(右)导数: 若导数定义中的”$\Delta$x->0” 改为”$\Delta$x->0+”(或”$\Delta$x->0-“)则为f(x)在l处的左(右)导数.
单侧导数: 左导数与右导数.
有限增量公式 : $\Delta$y = f^{‘}(x)$\Delta$x + a$\Delta$x a是$\Delta$x->0时的无穷小
幂指函数: 底数与指数都含有自变量的函数.

定理

  1. 连续是可导的必要不充分条件
  2. 可导函数的反函数, 四则运算, 复合操作得到的函数依然可导.

微分中值定理与导数的应用

微分中值定理

定义

极大值: f(x)在U(l)有定义, 若存在x∈U(l), f(x) $\leq$ f(l), 则f(l)是最大值
极小值: f(x)在U(l)有定义, 若存在x∈U(l), f(x) $\geq$ f(l), 则f(l)是最小值
极值: 极大值和极小值.
极值点: 使f(x)为极值的x称为极值点.
驻点: 若$f^{‘}(x)=0$, 则x为驻点.

定理

费马引理
: 若f(x)在x=a处取极值且可导, 则$f^{‘}(a) = 0$

利用f(a) > f(x), 构造导数定义, 后讨论左导与右导正负性, 利用可导等价左导等于右导, 得导数为0.

罗尔定理
: 若f(x)在[a, b]连续, (a, b)可导f(a) = f(b)则存在h∈(a, b)令$f^{‘}(h)=0$


闭区间连续使用介值定理得出有max与min, 端点相等得到max > f(a) = f(b)和min < f(a) = f(b)必据其一, 则(a, b)存在一个极值, 由可导推出导数为0.

拉格朗日中值定理
: 若f(x)在[a, b]上连续, (a, b)可导, 则存在c∈(a, b)使$f^{‘}(c) = “f(b) - f(a)” / “b - a”$

利用罗尔定理, 希望端点相等, 故考虑转动坐标系, 则得出g(x) = f(x) - “f(b) - f(a)” / “b - a” * (x - a), 然后利用罗尔定理证明

柯西中值定理
: 若f(x), g(x)满足[a, b]连续, (a, b)可导, g^{‘}(x) != 0, 则存在l∈(a, b) 使得f^{‘}(x) / g^{‘}(x) = “f(b) - f(a)” / “g(b) - g(a)”

将拉格朗日证明中的x, b, a换成g(x), g(b), g(a)

泰勒中值定理
: 若f(x)在x-=$x_0$处n+1阶可导, 则有$f(x) = f(x_0) + f^{(1)}(x_0)(x - x_0) + \frac{f^{(2)}(x_0)(x - x_0)^2}{2!} + … + \frac{f^{(n)}(x_0)(x - x_0)^n}{n!} + \frac{f^{(n+1)}(a)(x - x_0)^{n+1} }{(n+1)!}$

拉格朗日中值定理推论: 若f^{‘}(x)=0, x∈(a, b), 则f(x)=c, x∈(a, b)
积分准则: 导数相同的两个函数只差一个常数

泰勒公式

函数是无穷维的向量
线性组合: 将一个一般的向量表现为一组简单的向量的求和.

两种级数
傅立叶级数: $f(x) = c \times 1 + a_1sinx + a_2sin(2x)+……+a_nsin(nx) + b_1cos(x) + b_2cos(2x)+……+b_ncos(nx)$
幂级数: $f(x) = a_0 \times 1 + a_1(x - x_0) + a_2(x - x_0)^2 + …… + a_n(x - x_0)^n + ……$

解决三个问题, 得出泰勒公式

  1. 表示成幂级数的条件是什么?
    f(x)有$x - x_0$处可导, $f(x) = f(x_0) + A(x - x_0) + o[x - x_0]$

  2. 确定$a_0, a_1, a_2, …….$
    取$x = x_0$, 则$a_0 = f(x_0)$
    由幂级数的公式, 发现如果求一阶导, 则$a_0会消失, a_2, a_3, ……还会剩下(x - x_0)的结构使得取值后被消掉, 只剩下a_1$
    则$a_1 = f^{‘}(x_0)$
    由此得到$a_n = f^{(n)}(x_0) / n! $

  3. “+…”如何表示?
    皮亚诺公式 若f(x)在U($x_0$)中n阶可导, 则在此邻域内有f(x) = $f(x_0) + f’(x_0)(x - x_0) + “f’’(x_0)(x - x_0)” / 2! + … + f^{(x)(x_0)(x - x_0) / n!} + o[(x - x_0)^n] $
    证明:
    构造R(x) = f(x) - $(f(x_0) + f^{(1)}(x_0)(x - x_0) + “f^{(2)}(x_0)(x - x_0)” / 2! + … + f^{(x)}(x_0)(x - x_0) / n! + o[(x - x_0)^n] )$
    则只要证明$ lim_{x \rightarrow x_0}{R(x_0) / (x - x_0)^n} = 0$
    易知R(x)及n阶导数都为0
    则对极限用n-1次洛必达法则, 得原式 = $lim_{x \rightarrow x_0}{R^{(n-1)}(x) / “n!(x - x_0)”}$
    注意: 因为R(x)的n阶可导只能得出n-1阶连续, 则再用洛必达法则无法求出极限.
    此时应用柯西中值定理: $lim_{x \rightarrow x_0}{R^{(n-1)}(x) / “n!(x - x_0)”}$ = $”lim_{x \rightarrow x_0}{R^{(n-1)}(x) - R^{(n - 1)}(x_0)”/ “n!(x - x_0) - 0”}$ = $R^{(n)}(x_0) / n!$ = 0

积分

相关定理

微积分基本定理: 若f(x)在[a, b]上连续, 则$\exists c\in[a, b], \int^b_af(x)dx = f(c)(b - a)$.
压迫定理: 若f(x)$\leq$(或$\geq$)g(x), 则$\int f(x)dx \leq(或\geq)\int g(x)dx$.
原函数定理: 若f(x)在[a, b]连续, 则$\int^x_af(t)dt$在[a, b]可导且为f(x)的一个不定积分.
设F(x) = $\int^x_aF(t)dt$, 则F(x + $\Delta x$) - F(x) = $\int^{x+\Delta x}_aF(t)dt - \int^x_aF(t)dt = \int^{x+\Delta x}_xF(t)dt$, 则$\lim_{\Delta x \rightarrow 0}\frac{F(x+\Delta x) - F(x)}{\Delta x} = \lim_{\Delta x \rightarrow 0}\frac{\int^{x+\Delta x}_xf(t)dt}{\Delta x}$, 则上下都$\rightarrow 0$, 利用洛必达法则可证.
变上限定理: $(\int^{b(x)}_{a(x)}f(t)dt)^{‘} = f(b(x))b^{‘}(x) - f(a(x))a^{‘}(x)$.
参数方程积分: $设x = a(t), y = b(t), 则\int{ydx} = \int{b(t)a^{‘}(t)}$, 如果是定积分, 记得将上下限换成t的上下限.

广义积分

定义

无穷区间上: $\int^{\infty}{a}f(x)dx = \lim_{A\rightarrow\infty}{\int^A_af(x)dx} = \lim_{A\rightarrow\infty}{F(A)} - F(a)$
无界函数上:

  • 若$\lim_{x\rightarrow b^-}{f(x)dx} = \infty$或$\lim_{x\rightarrow a+}{f(x)dx} = \infty$, 则称$\lim{q\rightarrow0^+}{\int^{b-q}_a{f(x)dx} }$或$\lim{q\rightarrow 0^+}{\int^b_{a+q}{f(x)dx} }$为$\int^b_a{f(x)dx}$的广义积分.
  • 若$\exists$c$\in$[a, b], $lim_{x\rightarrow c}{f(x)} = \infty$, 则$\int^b_a{f(x)dx} = \int^c_a{f(x)dx} + \int^b_c{f(x)dx}$

相关定理

控制收敛定理: $\int^{\infty}_af(x)dx$收敛, $\exists$A$\in$(a, $\infty$), |g(x)| $\leq$ |f(x)|是$\int^{\infty}_{a}g(x)dx$收敛的充分不必要条件.
