Motion

质点运动

相关文字定义

位矢(r)
: 确定质点的位置的矢量.
速率(v)
: 速度的大小: $ \Delta s / \Delta t $
终端速度/收尾速度
: 考虑空气阻力的自由落体运动中物体达到的最大(也是最终)的速度.
切向加速度($a_l$)
: 沿轨道切线的加速度.
法向加速度(向心加速度)($a_n$)
: 沿轨道法线的加速度.
角加速度($\alpha$)
: 角速度与时间比值.

相关公式

平均速度: $ (\overline{a})=\Delta r / \Delta t $
瞬时速度: v=$\lim_{\Delta x \rightarrow 0}{\Delta r / \Delta t}=dr/dt $
加速度: a=$ \lim_{\Delta t \rightarrow 0}{\Delta v / \Delta t}=dv/dt=d^2v/dt^2 $

牛顿力学

相关公式

牛二: F=d(m v )/dt = m a
牛三: $F_{12}=-F_{21}$
胡克定律: f=-kx
万有引力: F=$Gm_1m_2\over r^2$
惯性力: 从加速参照系看静止参照系中的物体时, $F_0 = -ma_0$.

动量与角动量

相关定义

质心
: 与物体等效的质点.
质心运动定理
: 一个质点系的运动等效于一个质量等于质点系总质量的位于质心的质点c的运动, c所受的力等效于质点系所有外力和.

相关公式

动量(p): $Fdt = dp = dmv$
冲量(I): $I = \int_{t_1}^{t_2}{Fdt}$
质心表达式($r_c$): $r_c = \sum_im_ir_i/m = \int rdm / m$, $r_i$为质点到一个坐标中心的位矢.
$r_c$也可以分解
角动量(L): L = $r\times p = r \times mv$
力矩(M): M = $dL/dt = r\times F$, r为质点相对于固定点(常为转轴上的点或转动中心)的位矢.

功和能

相关定义

保守力
: 一对力所做的功只于质点初末位置有关的量.

相关公式

dW = $F \cdot dr$, r是位移.
$E_p = -Gm_1m_2/r$, r为公转半径.
$E_p = 1/2 * kx^2$

内力改变系统总动能, 但不能改变总动量.
保守力做的功等于势能减少量.

刚体的定轴转动

相关定义

刚体
: 受力是不改变形状体积的物体.

相关公式

角速度($\omega$): $\omega = d\theta/dt$
角加速度($\alpha$): $\alpha = d\omega / dt = d^2\theta / dt^2$
线速度(v): v = r$\omega$
切向加速度($a_l$): $a_l = r\alpha$
法向加速度($a_n$): $a_n = r\omega^2$
转动动能($E_k$): $E_k = J\omega^2/2$
定轴转动动能定理: $\int_{\theta_1}^{\theta_2}{Md\theta} = E_{k2} - E_{k1}$
刚体对转轴的角动量: $L_z = J_z\omega$, $M_z = J_z\alpha$$L_z, J_z$都是相对于转轴z来说的.
将运动学公式中的a换成$\alpha$, x换成$\theta$, v换成$\omega$, 就得到了转动公式
刚体转动定律: M = J$\alpha$
证明
考虑刚体绕一点转动, 取转轴为z轴, 考虑质元i
$\Delta$m表示质元的质量, $r_i$表示到转轴的距离, $a_i$表示其加速度, $F_i$表示受到的合力
取切向分量, $F_{ii} + f_{ii} = \Delta ma_{ii}, 利用a_{ii} = r\alpha$
则$F_{ii} + f_{ii} = \Delta mr_i\alpha$
因为切向分力同时垂直于半径和转轴, 故为消去内力乘上$r_i,再做累加.$
$\sum_i{F_{ii}r_i} + 1\sum_i{f_{ii}r_i} = \sum_i{\Delta m + r_i^2\alpha}$
因为内力成对出现, 考虑在同一垂直于转轴上的质元i, j之间的一对内力.因为沿同一直线上, 力臂相同, 而大小相反, 故力矩和为0
不在同一垂直于转轴的平面的两个质元的一对力可以分解为垂直转轴的分力, 这个分力大小相等方向相反.
所以左边第二项为0, 令右边$\sum_i{\Delta mr_i}$等于J, 则有M = J$\alpha$
平行轴定理: $J = J_0 + md^2$, $J_0$为过质心的轴的转动惯量, d为平行于前述轴的轴与前述轴的距离, J为不过质心的那个轴的转动惯量.

质点运动与刚体转动的对比

质点的运动 刚体的定轴转动
速度 $v = {dr}/{dt}$ 角速度 $\omega = d\theta/dt$
加速度 $a = dv/dt$ 角加速度 $d\alpha = d\omega/dt$
质量 m 转动惯量 $J = \int r^2dm$
力 F 力矩 $M = r \times F$
牛二 $F = ma$ 牛二 $M = J\alpha$
动量 p = mv 动量 $p = \sum_i \Delta mv_i$
角动量 $L = r \times p$ 角动量 $L = J\omega$
动量定理 $F = d(mv)/dt$ 角动量定理 $M = d(J\omega)/dt$
动量守恒, F = 0 角动量守恒, M = 0
力的功 $W_{ab} = \int_a^b F\cdot dr$ 力的功$W_{ab} = \int_{\theta_a}^{\theta_b} Md\theta$
动能 $E_k = mv^2/2$ 转动动能 $E_k = J\omega^2/2$

机械波

产生条件

  1. 波源
  2. 弹性介质

特点

  1. 机械振动在弹性介质的传播
  2. 介质中各质元在各自平衡位置附近震动
  3. 波动是能量或震动状态由近及远的传播.

分类

  • 横波: 震动方向与传播方向垂直, 如机械波.
  • 纵波: 质元震动方向和传播方向平行, 波形存在稀疏和紧密的区域, 如声波.
  • 其他波: 如水面波.

相关定义

波阵面
: 振动相位相同的点构成的平面.
波前
: 最前方的波阵面.
波线
: 表示波传播方向的有向线段.
波长($\lambda$)
: 沿波传播方向两相邻相同相位点之间的距离.
周期(T)
: 波前进一个波长距离所需的时间. 等于波源振动周期.
简谐波
: 简谐振动在介质中传播形成的波.
平面简谐波
: 波阵面为平面的波.
波函数($\xi$)
: 表示质元位移, 弹性介质形变, 气体压强等$\xi(r, t) = f(x, y, z, t)$.

相关公式

频率(v): v = 1/T
角频率($\omega$): $\omega = 2\pi v$, 只要看公式中t前的系数即可.
波速(相速)($\mu$): $\mu = \lambda / T = v\lambda$ 将cos内的东西设为常数, 然后把x对时间求导.
平面简谐波波函数: y(x, t) = Acos[$\omega (t \mp x/\mu) + \theta_0$]
沿X轴正向传播为-, 负向为+.
相差: $\Delta\theta = \frac{2\pi{x - x_0}}{\lambda}$
