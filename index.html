Nessus Crack

  1. Download the latest version
  2. Install it.
  3. Start the service and log in on https://localhost:8834.
  4. Choose Managed Scanner and continue.
  5. Choose Tenable.sc and continue.
  6. Set your username and password.
  7. After entering to the main page, close it.
  8. Stop the service.
  9. Execute /opt/nessus/sbin/nessuscli fetch --challenge to get the challenge code.
  10. Go to http://www.tenable.com/products/nessus-home for an active code.
  11. Go to https://plugins.nessus.org/offline.php to download the latest update package.
  12. Execute /opt/nessus/sbin/nessuscli update 升级包文件名.
  13. Execute find / -name 'plugin_feed_info.inc -print
  14. For all found files, add or change the variables like
    PLUGIN_FEED = "ProfessionalFeed (Direct)";
    PLUGIN_FEED_TRANSPORT = "Tenable Network Security Lightning";
  15. Execute
  16. Restart the service.
  17. Enjoy scanning.

2020-03-05

2020-03-05

好久没写日记了, 果然坚持一个需要付出精力的东西就是不容易啊. 如果不是今天晚上看到这个有意思的样本, 不知道还要再坑多久.

晚上写代码累了, 刷了一会儿知乎, 在孙杨问题下看到这个”气凝中华似朝阳”的人发表了一些非常没有逻辑的爱国护孙的言论, 像是一个典型的粉红. 我想找找乐子, 于是点进他的主页, 希望看点更搞笑的发言.

以前我一贯对于这种在证据确凿的问题下胡搅蛮缠毫无逻辑的人嗤之以鼻, 认为他们肯定是活在下层的卑微小人物, 整天在网上发泄愤怒. 但是我花了1个小时看他不多的回答记录, 发现了一些不一样的东西.

从他的回答可以看出, 他对中外古代近代历史的许多事件都有非常详细的了解, 比如对于太平天国运动, 他详细地讲述了曾国藩使用”结硬寨, 打呆仗”的战术击败太平军的过程. 通过引用关键的资料, 书籍, 论证昭和日本的投降首功在苏联而不是美国. 他的某些回答, 我必须承认, 如果平时随意刷到, 会给他点一个赞.

我陷入沉思: 一个读过不少书籍, 研究方向也是接近社会的历史事件, 为何在一个事情过程已经比较清楚的当下热点问题上犯下如此低级又没有逻辑的错误? 还执迷不悟?

接着, 我看是反省自己: 即使我读了很多的社会相关的书, 是不是依然会在遇到某些事件时被人诱导, 作出一些不恰当地判断与决定? 还有, 假如一个人随便引用了点什么东西让他显得专业, 我是否就会简单地对他的观点和结论照单全收?

为了更好地完善自己的判断体系, 我需要深入分析!

寻着他的回答, 我发现了一些蛛丝马迹. 大风起于青萍之末, 他的那些看似不错的回答里,也暴露了一些问题. 这些问题未必一定会导向对当下事件明显又固执的错误判断, 但我认为仍然具有很强的相关性.

首先, 他绝对相信自己的观点并带有一定的自恋性. 如果是被普遍认同感的观点, 这会被人认为是坚持真理, 可一旦论点出现问题, 他很难快速地开始分析并修正自己的观点, 反而容易为了维护自己的观点而进一步脱离实际, 造成观点的继续偏离, 而他在那些正确观点下的赞同, 更会强化这种倾向. 他坚信义和团的总体正面性, 遇到拿出证据反驳的人, 便胡搅蛮缠, 甚至会引用一些说法来攻击对方.

因此, 我应该在正常讨论中保持开放的态度, 面对友善的质疑时通过分析论证来维护观点. 同时, 不要因为几次正确就对自己的判断沾沾自喜, 面对任何事情都要慎重地作出独立的判断.

其次, 他几乎没有逻辑训练, 只从书中获得了一些零碎的片段. 结合自恋问题, 他对许多质疑相左的评论上来先进行攻击. 诸如” 你看待问题的方式很有问题”, “你没度过多少书就不要说话”.

我的逻辑训练也十分缺乏, 这会严重的影响我的判断能力. 进行系统的逻辑训练迫在眉睫.

还有, 他的所有回答都很情绪化, “谁知道?”, “谁想到?” 这样的语言层出不穷. 这样做也许能够调动观众感情进而获取支持, 但在个人化的事件分析中是非常容易造成问题的.

我需要将自己的情绪控制在合适的地方, 让他作出有益的贡献而不是干扰判断.

最后, 他对于民众(“民气可用”), 对于国家民族((所有中国史的回答都带有民族与国家的议论), 都有一种病态的信仰. 这导致了立场先行的问题, 尽管大量的阅读使他对于一些事件, 如义和团造成的破坏中, 有一些貌似公正的回答. 但因为立场已经严重地影响了他的判断, 为了证明义和团的总体正面性, 他强行说出” 看待一件事情不要看负面, 只要它对历史发展有推动作用就是正面的.” 这样强词夺理的话.

我应该时刻保持冷静, 不无条件的支持相信除了这句话以外的任何准则, 人事. 信仰是一件弊大于利的事物.

总结: 此人言论的巨大差异, 本质上是由于没有以合理的方式处理所阅读的内容, 缺乏必要的思考及联系只能重复书中的观点及论证, 致使无法从所学到的历史知识中总结更为普适的东西应用到生活中, 导致面对模糊复杂的现实问题时, 无法获得现成的正确的结论, 只能让一时的媒体舆论先入为主, 又被自恋限制了修正观点的能力, 最终导致明显的错误. 我们今后需要进行系统的逻辑训练, 增加对所获取的信息的个人化的分析处理并总结经验, 深入思考他人的论证而不是只取结论, 以便能为我们创造更好的生活.
