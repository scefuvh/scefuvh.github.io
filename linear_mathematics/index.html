行列式

相关定义

主对角线
: “\”方向的对角线.
次对角线
: “/“方向的对角线.
转置行列式
: 设 $ D=|a_{ij}| $ , 则 $ D^T = |a_{ji}| $ 即为转置行列式.
余子式($M^{(n-k)}$)
: 在n阶行列式D=|$a_{ij}$|中, 去掉第$i_1, i_2, …… i_n$行和第$j_1, j_2, … j_n$列交叉点所有的元素$a_{ij}$, 剩下的元素按原来的相对位置构成的行列式叫做D中k阶子式$N^{(k)}$的余子式.
代数余子式($A^{(n-k)}$)
: $A^{(n-k)} = M^{(n-k)}(-1)^{i_1+i_2+…+i_n+j_i+j_2+…+j_n}$
子式($N^{(k)}$)
: 在n阶行列式中选取k行k列, 将交叉点处的元素按原来的相对位置组成一个新的行列式, 成为D的一个k阶子式.
拉普拉斯定理
: 在n阶行列式D中给定的k行的所有的子式乘上对应的代数余子式的和等于D的值.

计算方法

二阶: 主对角元乘积 - 次对角元乘积.
三阶: 穿针法.
n阶: 从1-n行, 每行选取一个与之前所选元素不重复的列的元素相乘, 将所有可能相加.
定义公式: $ D=|a_{ij}|_n=(-1)^{\tau(i_1i_2…i_n)} + \sum_{k_1k_2k_3…k_n}(-1)^{\tau(k_1k_2…k_n)}a_{i_1k_1} a_{i_2k_2} a_{i_3k_3} … a_{i_nk_n} $, $(i_1i_2i_3…i_n)$为一个给定的排列.
实际操作: 利用含有1的列不断将同列其他元素变为0后再展开依次降阶.

相关性质

  • 行列式具有对称性.
  • $ D^T = D $
  • 交换行列式中任意两行, 行列式的值变为相反数.
  • 行列式中两行成比例, 则行列式值为0.
  • 行列式中一行元素乘k, 行列式值乘k.
  • 若行列式存在0行, 值为0.
  • 若行列式中一行的每个元素都是两个元素$a_{ij}, b_{ij}$的和, 则行列式值等于分别用$a_{ij}和b_{ij}$替换该行元素的两个新行列式的值的和.
  • 行列式中一行元素乘上k后加到另一行对应列元素上, 行列式值不变.

逆矩阵

相关定义

逆矩阵
: 对于n阶方阵A, 如果存在n阶方阵B, 使AB=BA=E, 则A,B互为逆矩阵, 记为$A^{-1}$.
奇异矩阵(不可逆矩阵/满秩矩阵)
: 若|A|=0, 则|A|是奇异矩阵.
非奇异矩阵
: 若|A| != 0, 则|A|是非奇异矩阵.
伴随矩阵($A^*$)
: 对n阶方阵B, |B|的各个余子式按$A_{ij} = a_{ji}$的格式构成的行列式为A的伴随矩阵$B^*$.

定理

  • A的逆矩阵最多一个.
  • 设A为n阶方阵, A可逆的充要条件是|A| != 0, $A^{-1}=\frac{A^{*} }{|A|}$
  • 对于n阶方阵$A_1A_2…A_k = nE, 且n != 0$, 则$A_1$到$A_k$都可逆.

运算性质

  • 证明
    若$|A| != 0, A^{}A = |A|E, 两边取行列式得|A||A^{}| = |A|^n$.
    若$|A|=0$, 则易证.
  • 对角矩阵A的逆矩阵即对角元倒数组成的矩阵.
  • 二阶矩阵快速求法 主对调, 副变号, 再处以矩阵对应的行列式

初等变换求矩阵的逆矩阵

因为任何可逆矩阵都可经过初等 行变换为单位矩阵, 所以A = PE, P是一系列初等矩阵的乘积, 所以$A^{-1} = P^{-1}$, 若$P = P_1P_2P_3…P_n, 则A^{-1} = P_n^{-1}…P_3^{-1}P_2^{-1}P_1^{-1}E$, 结合A = PE得到的$P^{-1}A = E$. 利用分块矩阵, $P^{-1}(A E) = (E A^{-1})$.
那么, 只要构造n$\times$2n阶方阵(A E)并对其做初等行变换至A变为E, 则E就变为$A^{-1}$; 若无法达成, 则A不可逆.
类似, AX = B的方程可以将(A B)初等行变换成(E $A^{-1}B$)来解决.
XA = B的方程可以通过转置变成AX = B形式.

n维向量

相关定义

n维向量
: n个数组成的有序数组. 默认列向量.
分量
: 向量中的任意一个数.
零向量
: 所有分量全部为0的向量.
向量相等
: 每个分量都相等的同维向量相等.
线性组合(线性表示)
: 对n维向量$a_1, a_2 … a_n, b$若存在数$k_1, k_2, … ,k_n$使得$b = k_1a_1 + k_2a_2 + … + k_na_n$则称b是向量$a_1, a_2 … a_n, b$的一个线性组合, 这些k为组合系数或称表示系数.
向量组等价
: 多个彼此可以互相线性表示的向量组.
线性相关
: 若存在不全为0的实数$k_1, k_2 \cdots k_n$使$a_1, a_2 \cdots a_n = 0$, 则称这些向量线性相关.
截短向量与接长向量
: 从原始向量的一部分为原始向量的截短向量, 原始向量是那一部分的接长向量.
极大线性无关组(极大无关组)
: 设($b_1, b_2, \cdots b_n$)是向量组$a_1, a_2, \cdots a_n$的一部分, 若b向量组线性无关且a向量组的每一个向量都可以由b向量组线性表示, 则称b组是a组的极大无关组.
向量组的秩
: 零向量组的秩为0, 非零向量组的秩为其极大无关组的向量个数.
矩阵的列(行)秩
: 矩阵A的列(行)向量组的秩称为矩阵的列(行)秩.

相关定理

  • $a_1, a_2, \cdots a_n$与b线性相关的充要条件是其对应的线性方程组有解.
  • $b_1, b_2, \cdots b_n$可由$a_1, a_2, \cdots a_n$线性表示的充要条件是以$a_1, a_2, \cdots a_n$为列构造的矩阵与$a_1, a_2, \cdots a_n, b_1, b_2, \cdots b_n$为列构造的矩阵的秩相等.
  • n维向量$a_1, a_2, \cdots a_n$线性相关的充要条件为对应的齐次线性方程组有非零解, 也即|($a_1, a_2, \cdots a_n$)| = 0或r($a_1, a_2, \cdots a_n$) $<$ 未知数个数或$a_1, a_2, \cdots a_n$的维数$<$n.
  • $a_1, a_2, \cdots a_n$线性相(无)关, 则其中任何部分的向量之间也线性相(无)关.
  • 若一些向量线性无关, 则他们的接长向量也线性无关.
  • 一个向量组的所有极大无关组与该向量组等价.
  • 向量组$a_1, a_2, \cdots a_n$有极大无关组的充要条件是向量组里存在非零向量.
    因为含有零向量矩阵一定线性相关, 则必要性得证.
    下面证明充分性, 设a向量组中至少一个非零向量, 则从它不能表示的向量组中的向量中挑一个与它组成新的向量组, 则这两个向量线性无关, 递归以上过程直到出现极大无关组.
    因为以上过程一定可行, 则充分性可证

    欲证明此事, 只要有不全为零的数$k_1, k_2, \cdots k_s$, 令$\sum_{i=1}^{s}{k_ia_i} = 0$即可
    设$a_i = \sum_{i=1}^{t}{l_{ji}b_j}, i\in[1, s]\in N^+$, 则

    合并以上s个等式.

    设C为右边的矩阵, Cx = 0
    则因为s>t, r(C) = r($\breve C$) $<$ s, 故上面的方程组有非零解.
    即存在$k_1, k_2, \cdots k_s$, 使得C$(k_1, k_2, \cdots k_s)^T$ = 0.
    因为$\sum_{i=1}^{s}{k_ia_i} = (a_1, a_2 \cdots a_s)(k_1, k_2, \cdots k_s)^T = (b_1, b_2 \cdots b_t)C(k_1, k_2, \cdots k_s) = 0$.

    • 逆否命题: 若a向量组线性无关, 可由b向量组表示, 则a向量组的向量个数$\leq$b向量组的向量个数.
    • 推论: 两个线性无关的等价向量组必有相同向量个数.

解线性方程组

方程组形式

矩阵形式: Ax = b, 其中A是一个矩阵, x是一个列向量(又称解向量), b是一个列矩阵.
向量表示: $x_1\alpha_1 + x_2\alpha_2 + … + x_n\alpha_n = \beta$.
其中$\alpha_i = \left[
\begin{matrix}
a_{1i} \ a_{2i} \ a_{3i} \ \vdots \ a_{ni}
\end{matrix}
\right]$, $\beta$为等式右边常量组成的列向量.

相关定义

齐次线性方程组
: 等式右边的常数为0的方程组.
同解方程组
: 解集相同的多个方程组.
自由未知量
: 用以表示其它未知量, 而自己没有确定取值且未被其他未知量表示的未知量.
基础解系
: 设$a_1, a_2 \cdots a_n$为一组解向量, 若他们线性无关且可以线性表示任意一个解向量, 则称他们为方程组的一个基础解系, 本质上是向量组的极大无关组.
导出组
: 对一个非齐次线性方程组, 另右边常数项为0得到的齐次线性方程组为那个非齐次线性方程组的导出组.

相关定理

齐次线性方程组

  • 设线性方程组Ax = B, 当r(A) = r($\breve{a}$) = n时, 方程组有唯一解, 若r(A) = r($\breve{A}$) $<$ n, 方程组有无穷多解, 若r(A) != r($\breve{A}$), 方程组无解.
  • 因为解向量的线性组合也是齐次线性方程组的解, 所以含有非零解的齐次线性方程组有无穷多解.
  • 经过初等变换的方程组的解与原方程组的解相同.由此可得出齐次线性方程组总有解, 且若方程个数小于未知量个数则必有非零解.
  • 若r(A) $<$ 未知数个数n, 则任意n-r(A)个解向量组成的的向量组是一个基础解系.

非齐次线性方程组

  • 若a, b是两个解向量, 则a-b为导出组的一个解向量.
  • 若y是一个解向量, n是导出组的一个通解, 则y+n也是一个解向量.
  • 任意解向量可以表示为特解y+导出组的某个解向量n.
  • 若方程组满足r(A) = r(A, b), n是导出组的基础解系, y是一个特解, 则方程组的通解为y+n.

高斯消元法

设未知数个数为n.
考虑线性方程组AX = B(A, B是矩阵),则称C = (A, B)为这个方程组的增广矩阵.
如果B中所有元素为0, 为齐次线性方程组.
若将每个x的代入方程组中成立的取值按顺序组成一个列向量, 则称其为那个方程组的解向量.
方程组的高斯消元法本质上等价于对其增广矩阵进行初等行变换到规范的阶梯形矩阵.
化解增广矩阵时, 如果最下面一个常数元素无法化成0, 则原方程无解.
如果那个常数元素为0, 但是矩阵的秩小于未知数个数, 则方程有无数多解. 若相等, 方程有唯一解.

Cramer Theorem

n元线性方程组系数行列式D不为0是此方程组有唯一解的 充分不必要条件.

其中$D_j$为将D中j列换成方程组右边常数的行列式.
若齐次线性方程组的D不为0, 则其只有一x全为0的解. 反之, 可能有非零解.

: 设P为一个包含0, 1的数集, 若P中数的和差积商都在P中, 则成P是一个数域.
维数
: 设V是数域P上的线性空间, 若$a_1, a_2, \cdots, a_n$是V中n个线性无关的向量, 且V中任意i向量都可以由他们表示, 则称V为n维线性空间.类似极大无关组, 注意$R^n$的线性子空间不一定是n维向量.
基
: 维数定义中$a_1, a_2, \cdots, a_n$就叫V的一组基.
坐标
: 任何数域P上的线性空间V中的任何一个元素y都可以写成$y = (a_1, a_2 \cdots a_n)\left(
\begin{matrix}
b_1 \ b_2 \ \vdots \ b_n
\end{matrix}
\right)$其中a向量组是一组基, 所以可以将线性空间中的元素称为向量, b向量组称为坐标.
同构
: 设V和K都是实线性空间, f是V到K的双射若a, 若f和同一项中的常数在运算中可以互换, 则V和K同构.
过渡矩阵
: 设$C_n$, a,b向量组是n维线性空间V的两组基, 若有b = a$\times$C, 则称C是基a到基b的过渡矩阵.
欧氏空间
: 定义了内积的实线性空间.
长度
: 设V是欧式空间, 则a$\in$V的长度为$\sqrt{(a, a)}$
a的单位化
: 通过$\frac{a}{|a|}$得到的单位向量.
夹角
: 设V是欧式空间, 对非零的a, b$\in$V, 定义a, b的夹角$<$a, b$>$ = arccos$\frac{\}{|a||b|}$
正交
: 元素a, b垂直时.
距离
: 欧式空间v中元素a, b之间的距离为|a-b|
正交向量组
: 若欧式空间v中一组非零向量$a_1, a_2 \cdots a_n$两两正交, 则称为一个正交向量组
正交基
: 两两正交的一组基.
标准正交基
: 所有向量都是单位向量的正交基.

运算定义

设p是一个数域, v是一个非空集合.
加法
: 如果对任意两个元素a, b$\in$v, 总有唯一的y$\in$v与之对应, 则称y是a与b的和.
数乘
: 如果对于任一数k$\in$p与与任意元素a$\in$v, 总有唯一元素d$\in$v与之对应, 则称d为ak的的积.
线性空间
: 满足以下8条运算律, 且对于加法和数乘封闭的空间称为线性空间.

  1. a + b = b + a.
  2. a + b + c = a + (b + c)
  3. V中存在元素0, 使得$\forall$a$\in$V都有a + 0 = a
  4. 对任何a$\in$V, 都有a的负元素b, 使得a + b = 0
  5. 1a = a
  6. abc = a(bc)
  7. a(b + c) = ab + ac
  8. (a + b)c = ac + bc
    线性子空间
    : 线性空间V中仍为线性空间的一个V的子集.
    生成的线性子空间
    : 设V为P上的线性空间, $a_1, a_2, \cdots a_n \in V$, 则L($a_1, a_2 \cdots a_n$) = {$k_1a_1 + k_2a_2 \cdots k_na_n \in V | k_i \in P|$}是V的子空间, 成为由$a_1, a_2, \cdots a_n$生成的子空间.
    内积
    : $\exists a, b\in V$, 记(a, b) V是一个实线性空间, 若存在唯一实数与他们对应, 且满足
    1. (a, b) = (b, a)
    2. (ka, b) = k(a, b)
    3. (a+b, c) = (a, y) + (b, y)
    4. (a, a) $\geq$ 0, 当且仅当a = 0时取=
    • 在$R^n$中, 定义内积为a,b对应元素乘积的和.
    • 在实线性空间$M^{m\times n}$中, 定义A = $(a_{ij})_{m\times n}$, $B = (b_{ij})_{m\times n}$的内积为($\sum^m_{i=1}\sum^n_{j=1}a_{ij}b_{ij}$)
    • C[a, b]中对任意f(x), g(x)可定义内积为$\int^b_af(x)g(x)dx$
    • 标准正交基下元素的内积和长度计算等价于对应坐标向量的计算.

线性空间的性质

  • 负元素和零元素唯一
  • 0a = 0
  • a0 = 0
  • -1a = -a

相关定理

  • 线性空间V的子集L是线性子空间的充要条件是L对V中的线性运算封闭.
  • V与K同构, f是V到K的双射, 则V中向量组$a_1, a_2 \cdots a_n$线性相关的充要条件的是f($a_1$), f($a_2$) $\cdots$ f($a_n$)线性相关, 否命题也成立.
  • 同构的实线性空间有相同的维数.
  • 任何一个实n维线性空间与$R^n$是同构的.
  • 因为同构映射保持了向量之间线性关系的相同, 所以讨论V中元素关系就可以讨论坐标向量的线性关系来进行.
  • 设C是基a到基b的过渡矩阵, 则C的第j列就是$b_j$在a下的坐标.
  • 设C是基a到基b的过渡矩阵, 则C的列向量组线性无关, C是满秩.
  • 设C是基a到基b的过渡矩阵, 若元素k在a下的坐标为向量组$a_1$, 在b下的坐标为$a_2$, 则$a_2 = C^{-1}a_1$.
  • 设a, b是欧式空间V内的元素, k是任意实数, 则:
    • |a| $\geq$ 0, 当且仅当a=0是取等号.
    • |ka| = k|a|
    • |(a, b)| $\leq$ |a||b|, 当且仅当a, b线性相关时成立

常见的线性空间

  • 全体n维向量.
  • 全体m$\times$n矩阵.
  • 次数小于n的全体多项式(记为$R[x]_{<n}$)
  • 正弦函数的集合.
  • 齐次线性方程组的全体解向量.
  • [a, b]上的全体实连续函数.

Gram-Schmidt正交化法

设$a_1, a_2, \cdots a_n$是欧式空间B的一组基.
令$b_1 = a_1$
按照$b_i = \sum^n_{i=2}{\frac{(b_{i-1}, a_i)b_{i-1} }{b_{i-1}, b_{i-1} }}$
单位化$c_i = \frac{b_i}{|b_i|}$
得到与a向量组等价的正交基c. , 且两边对应的部分也等价.

矩阵的对角化

相关定义

特征值
: 设A是数域P上的n阶方阵, 若存在数k$\in$P和P上的n阶非零列向量a使得Aa = ka, 则称k是A的特征值, a是A对应k的特征向量.
特征子空间($V_k$)
: 设A是数域P上的n阶方阵, k是A的特征值, 则{A对应于k的所有特征向量}$\cup${0}是(kE-A)X = 0的解空间, 称为A对应于k的特征子空间.
特征多项式
: 设A是数域P上的n阶方阵, k是A的特征值, 则f(k) = |kE - A|为A的特征多项式.
特征方程
: 将特征多项式取值为0得到的关于k的方程.
迹
: 设n阶方阵A$=(a_{ij})_n$, 则$\sum^n_{i=1}{a_{a_{ii} }}$成为A的迹, 记作tr(A)
代数重数
: 若对于方阵A, k为n重特征值, 则n为k的代数重数.
几何重数
: 设k为A的特征值, r为$V_k$的维数, 则称r为k的几何重数.
相似矩阵
: 设A, B为n阶方阵, 若$\exists$P, 使得$P^{-1}AP = B$, 则称A, B相似.
正交矩阵
: 若$AA^T = E$, 则其为正交矩阵.

相关定理

  • n阶方阵A的特征值即(kE - A)X = 0有非零解的k值.
  • 设k是方阵A的一个特征值, $a_1, a_2 \cdots a_s$是对应k的特征向量, 任取$k_1, k_2 \cdots k_s$,, 若$\sum_{i=1}{s}{k_ia_i} \neq 0$, 则其也是对应于k的特征向量.
  • 一个特征向量对应唯一特征值, 但一个特征值可以对应多个特征向量.
  • n阶方阵A有n个特征值(可以相等), 且有
    • $\sum^n_{i=1}{k_i} = \sum^n_{i=1}a_{ii} = tr(A)$
    • $\prod^n_{i=1}{k_i} = |A|$
  • 对应与不同特征值的特征向量之间线性无关.
    • 设$k_1, k_2 \cdots k_n$是方阵A的n个互异特征值, 则$a_{i1}, a_{i2} \cdots a_{ir_i}$是对应于$k_i$的特征向量, 则$a_{11}, a_{12} \cdots a_{1r_1} \cdots a_{21}, a_{22} \cdots a_{2r_2} \cdots a_{nr_n}$线性无关.
  • $A^T$的特征多项式与$A$的特征多项式相同.
  • 设k为A的特征值, a为A对应于k的特征向量, 则有$qk$为kA对应于a的特征值, $k^q$为$A^q$对应于a的特征值, g(A) = $a_0A^q + a_1A^{q-1} + \cdots a_qE$, 则g(k)为g(A)对应于a的特征值, q$\in$C, $a_i\in$C.
  • $k^{-1}$为$A^{-1}$的特征值, 特征向量不变.
  • $\frac{|A|}{k}为A^*$的特征值, 特征向量不变.
  • 如果方阵的一个特征值k为n重特征值, 则$V_k$的维数$\leq$n.
  • 一个n阶方阵的代数重数和为n.
  • 相似矩阵具有自反性, 对称性, 传递性.
  • 若A~B, 则|kE - A| = |kE - B|.
    • kA ~ kB
    • $A^m$ ~ $B^m$
    • g(A) ~ g(B), g的定义见上.
  • 存在对角矩阵B, 使n阶方阵A与B相似的充要条件是A有n个线性无关的特征向量, 且这个存在的B的主对角元为A的每个特征值.
  • n阶方阵A相似对角矩阵的充要条件是A的每个特征值的代数重数等于几何重数.
  • A是正交矩阵的充要条件是A的列向量是两两正交的单位向量组.
  • 若A是正交矩阵, 则$A^{-1} = A^T$
  • 若A, B是正交矩阵, Ab也是正交矩阵.
  • 正交矩阵的实特征值只能为1或-1.
  • 对称矩阵的特征值与特征相量为实数.
  • 对对称矩阵A, 若k $\neq$ h, 特征值k对应的向量a与特征值h对应的向量b正交.
  • 实对称矩阵每个特征值的代数重数等于几何重数.
    • 设A为实对称矩阵, 则必定存在正交矩阵Q使得$Q^{-1}$AQ为对角矩阵.

求特征值与特征向量

特征值: 求出特征方程的所有解$k_1, k_2 \cdots k_n$.
特征向量: 对于特征值$k_i$, 求出$( k_iE - A )X = 0$的基础解系, 则那个基础解系就是$k_i$对应的特征向量.

判断并求出n阶方阵A相似的对角矩阵

(1): 求出A的所有特征值, 设A有s个不同的特征值k, n为对应的代数重数.
(2): 对每一个特征值$k_i$, 验证r($k_iE - A$) = n - $n_i$, 若任何一个为否, 则A无相似的对角矩阵.
(3): 对$k_i$, 求$V_{k_i}$的基, 把这些基作为列向量构造P, 则按照列向量与特征值一一对应的顺序排成的diag(k)即为所求.

实对称矩阵A的对角化

(1): 求出A的特征值.
(2): 将每个特征值对应的特征向量正交单位化.
(3): 将它们组合起来组成Q.
(4): 将特征值按照Q中特征向量对应的顺序写成对角矩阵.
